## 📝 **Description**

**Peticare** is a modern, playful, and beautifully crafted mobile UI kit designed for pet stores, clinics, grooming, and pet care-related services. Built entirely with Flutter, this cross-platform kit provides a scalable and responsive foundation for mobile app development.

Perfect for developers looking to kickstart a pet care app or agencies creating client solutions, Peticare blends friendly visuals with reusable code components.

## 🌟 **Features**

- 🔨 Built 100% with Flutter (no third-party templates)
- 📱 +50 Ready-to-Use Screens
- 🧩 Modular and reusable widgets
- 🎨 Clean, responsive layouts
- 🎥 Includes Smooth animations, Icons, and Assets
- 🌙 Light & dark mode support

## 📸 **UI Screens Overview**

### 🐕 Onboarding

- Welcomes users and introduces Peticare’s main features with clean visuals.

### 👤 Authentication

- Minimal, form-focused authentication for a smooth start.

### 🏠 Dashboard

- Shows daily tasks, reminders, and quick pet insights.

### 🐾 Pet Profile

- Displays pet info, health stats, and activity overview.

- Tracks feeding, grooming, and medical routines.

### 📆 Appointments

- Manages vet and grooming visits with clear status cards.

### 🛒 Store

- Shop for your pet’s essentials with an intuitive and visually clean shopping experience.

- A sleek, user-friendly store layout designed for effortless product discovery and checkout.

### ⚙️ Profile & Settings

- Manages themes, notifications, and account preferences.

### ✨ More

- 🔔 Notifications: Keeps users updated with reminders and upcoming events.

- 📸 Gallery: Stores memorable pet moments with organized photo collections.

- 📅 Activity History: Displays past health logs, tasks, and appointment summaries.

- 💬 Tips & Insights: Provides quick, helpful advice for better pet care

## 🚀 **Getting Started**

1. Download the project folder.
2. Make sure you have Flutter installed.

   you can check with running the command:

   ```bash
   flutter doctor
   ```

3. Run to fetch dependencies by running.
   ```bash
   flutter pub get
   ```
4. Run the app
   ```bash
   flutter run
   ```
   (Make sure a simulator/emulator or physical device is connected.)
5. Explore each module inside `/lib/screens/` and `/lib/widgets/`.

**Requirements:**

- Flutter 3.x or later
- Dart SDK
- Android Studio / VS Code
